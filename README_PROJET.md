# Rapport CI1 Leroux

Ce dépôt sert à préparer le rapport d’activité entreprise CI1 de l’étudiant dans le cadre de son alternance chez Leroux au service informatique.

Le projet est organisé en fichiers Markdown afin de travailler progressivement :

- contexte ;
- plan ;
- rédaction ;
- annexes ;
- sources ;
- contrôle qualité ;
- export final.

Les fichiers du dossier `00_contexte_source` sont la source de vérité.
Aucune rédaction finale ne doit être produite tant que les fichiers de contexte et le plan ne sont pas validés.
