# Style d’écriture attendu

Le rapport doit être personnalisé, fidèle à l’expérience réelle de l’étudiant, à ses missions en entreprise et à sa manière habituelle d’expliquer les choses.

Le style doit rester proche de ses anciens rapports, tout en améliorant :

- la structure ;
- la clarté ;
- l’orthographe ;
- la syntaxe ;
- la qualité rédactionnelle.

## À conserver

- écriture explicative ;
- progression logique ;
- contextualisation avant les détails techniques ;
- première personne pour les missions ;
- ton professionnel mais accessible ;
- bilan personnel clair.

## À corriger

- fautes ;
- répétitions ;
- phrases trop longues ;
- formulations maladroites ;
- transitions trop lourdes ;
- passages trop génériques.

## À éviter

- style trop académique ;
- vocabulaire excessivement soutenu ;
- survalorisation ;
- invention ;
- phrases qui ne correspondent pas au niveau réel de l’étudiant.
