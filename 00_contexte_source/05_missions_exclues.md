# Missions exclues comme missions principales

Les sujets suivants ne doivent pas être utilisés comme missions principales :

- Xibo ;
- uniFLOW ;
- GLPI.

## Raison

Ces sujets sont considérés comme incomplets pour porter une analyse principale dans le rapport CI1.

## Utilisation autorisée

- citation courte ;
- contexte ;
- outil ou activité ponctuelle ;
- élément secondaire.

## Utilisation interdite

- mission principale ;
- partie complète ;
- conclusion centrée sur ces sujets ;
- analyse centrale du rapport.
